## Overview

A real-time 2D fluid dynamics simulator that solves the incompressible Navier-Stokes equations on the GPU using WebGL compute shaders.

## Motivation

I wanted to understand the mathematics behind fluid simulation and explore GPU programming techniques for scientific computing.

## Implementation

The simulation uses a staggered MAC grid and a pressure projection method to enforce incompressibility. The entire solver runs on the GPU via WebGL 2.0.

### Performance

- 512×512 grid at 60 FPS on a mid-range GPU
- Pressure solver converges in ~20 Jacobi iterations per frame
