## Overview

This project involved designing and building a 6-degree-of-freedom robotic arm from scratch, capable of performing autonomous pick-and-place operations in an unstructured environment.

## Technical Details

The arm uses a combination of servo motors and custom 3D-printed joints. The control system runs on a Raspberry Pi 4 and uses OpenCV for object detection.

### Key Features

- Real-time object detection using YOLOv8
- Inverse kinematics solver implemented in Python
- Custom PCB for motor control
- ROS2 integration for modularity

## Results

The arm achieved a pick-and-place success rate of 94% in controlled testing conditions.
